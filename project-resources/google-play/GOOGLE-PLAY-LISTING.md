## â™¿ Accessibility Declaration

### Declaration Questions & Answers

**Q: Does your app use the Accessibility API?**
```
Yes
```

**Q: Why does your app need accessibility permissions?**
```
This app provides a software power button replacement for users with 
broken or hard-to-use physical power buttons. The Accessibility Service 
is used to perform screen lock and screen-off actions when the user 
taps the floating power button. This is the core functionality of the 
app and directly assists users with accessibility needs.
```

**Q: What accessibility features does your app provide?**
```
- Floating on-screen button for users who cannot use physical power button
- One-tap screen lock without needing hardware button
- Screen off functionality for users with motor disabilities
- Large, customizable touch targets
- Alternative to physical button interaction
```

**Q: Does your app use accessibility to read screen content?**
```
No. The app explicitly sets canRetrieveWindowContent="false" in the 
accessibility service configuration. No screen content is ever read, 
captured, or processed.
```

**Q: Does your app collect data using accessibility?**
```
No. The Accessibility Service is used only to perform user-initiated 
actions (lock screen, turn off display). No data is collected, stored, 
or transmitted.
```

### FAQ (for support responses)

**Q: Why can't the app shut down my phone?**
```
Android does not allow apps to shut down or restart devices for security 
reasons. This is a platform limitation. Our app provides alternatives 
for the most common power button uses: locking the screen and turning 
off the display.
```

**Q: Why does the app need Accessibility permission?**
```
The Accessibility permission allows the app to perform the lock screen 
action when you tap the floating button. This is the only way Android 
allows apps to lock the screen. We don't read any screen content or 
collect any data.
```

**Q: The floating button disappeared. How do I get it back?**
```
Open the app and toggle the floating button switch on the home screen. 
If that doesn't work, check that the "Draw over other apps" permission 
is still enabled in your device settings.
```

**Q: Does this app work when the screen is off?**
```
The floating button is visible when your screen is on. To turn the 
screen back on with a broken power button, you can use features like 
double-tap to wake, fingerprint sensor, or scheduled alarms.
```
